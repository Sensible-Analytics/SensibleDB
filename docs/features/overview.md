# Features Overview

## Built-in MCP Tools

AI agents can discover data and walk the graph autonomously.

## Built-in Embeddings

Use Embed() directly in queries - no external service needed.

## RAG Tooling

Vector search, BM25 keyword search, graph traversals, hybrid search with RRF.

## Security

Private by default. Data accessible only through compiled NQL queries.

## Ultra-Low Latency

Rust + LMDB memory-mapped B-trees for near-zero overhead access.

## Type-Safe Queries

Compile-time validation catches errors before production.

## Multi-Model

Graph, vector, KV, document, and relational - all in one database.
